Flower Store API

Modern REST API for flower store management

System Online - All Services Running

Quick Start

Try the API endpoints below. Full documentation available in GitHub README.

Flower Endpoints

GET /api/v1/flower
Get all flowers from catalog
POST /api/v1/flower
Add a new flower
GET /api/v1/flower/type/{type}
Get flower by type (ROSE, TULIP, etc.)

User Endpoints

GET /api/v1/users
List all registered users
POST /api/v1/users
Register new user (auto-calculates age)

Order & Services

POST /api/orders
Create order with delivery & payment
GET /api/v1/delivery
List delivery methods (Post, DHL)
GET /api/v1/payment
List payment methods (Card, PayPal)

Database Info

PostgreSQL Database with 2 tables:

flowers - id, type, color, sepal_length, price, description

app_users - id, email (unique), dob, age (calculated via @Transient)

Available flower types: ROSE, TULIP, ROMASHKA, CACTUS
Available colors: RED, GREEN, BLUE, WHITE, YELLOW
